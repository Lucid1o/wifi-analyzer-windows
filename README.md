# WiFi Analyzer (Windows)

This is a simple WiFi Analyzer application built using Python and Tkinter.

## Features
- Scan available WiFi networks (Windows)
- Test network quality using ping
- Calculate packet loss and latency
- Display a basic network rating
- Provide suggestions to improve WiFi performance

## Requirements
- Windows OS
- Python 3.x
- Wireshark not required
- Libraries: ping3

## How to Run
1. Install Python
2. Install required library:
   pip install ping3
3. Run the application:
   python wifi_analyzer.py

## Purpose
This project was created as a learning exercise for understanding basic networking concepts and GUI development using Python.
